# Food Recipe Search Engine

Discover recipes powered by [TheMealDB](https://www.themealdb.com/) with category filtering, detailed ingredient breakdowns, and a responsive design built on Tailwind CSS.

## Getting Started

1. Open `index.html` in your browser.
2. Search by name, filter by category, or explore the default popular recipes.
3. Click any recipe card to view detailed instructions and ingredients.

## Project Structure

- `index.html` – Main page and layout.
- `styles/global.css` – Custom fonts and scrollbar styling.
- `scripts/app.js` – Client-side logic, API calls, and UI interactions.

## Features

- Dynamic search with API-backed suggestions.
- Category filters populated at runtime.
- Detailed recipe modal with ingredients and instructions.
- Exponential backoff for robust API requests.
